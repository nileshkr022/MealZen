# 🥗 MealZen — Your AI-Powered Kitchen Companion

**MealZen** is a full-stack **MERN** application crafted to bring calm and clarity to your kitchen!  
Think of it as your **AI sous-chef** — helping you track ingredients, avoid food waste, and discover delicious personalized meal plans tailored to your pantry and goals.  

---

## 🚀 Live Demo
👉 [**mealzenbynilesh.netlify.app**](https://mealzenbynilesh.netlify.app)

---

## ✨ Core Features

### ✅ Smart Pantry Management  
Keep tabs on your ingredients, monitor expiry dates, and get timely alerts before things go bad.

### 🤖 AI-Powered Recipe Generator  
Generate unique and mouthwatering recipes using only what’s available in your pantry.

### 🍽️ Personalized 7-Day Meal Plans *(Pro Feature)*  
Get AI-crafted meal plans based on your taste preferences, dietary goals, and available ingredients.

### 💳 Pro Subscription System  
Unlock premium features securely with **Razorpay** integration.

### 🔐 Role-Based Authentication  
Secure **JWT-based** authentication system for Free Users, Pro Users, and Admins.

### 🛠️ Admin Dashboard  
Manage users, track activity, and moderate content using a powerful admin interface.

### 🧾 PDF Meal Plan Export  
Pro users can export their weekly meal plans as clean, printable PDFs using **PDFKit**.

## 🧩 Tech Stack

### 🎨 Frontend
- **React.js** — Modern UI library  
- **React Router** — Seamless page navigation  
- **Tailwind CSS** — Utility-first styling framework  
- **React Context API** — Global state management  
- **Axios** — Promise-based HTTP client  
- **Vite** — Lightning-fast build tool  

### 🧰 Backend
- **Node.js + Express.js** — RESTful API development  
- **MongoDB + Mongoose** — NoSQL database and schema modeling  
- **JWT** — Secure authentication mechanism  
- **Nodemailer** — Send welcome/reset/notification emails  
- **PDFKit** — Generate PDF documents server-side  
- **OpenRouter API** — Access advanced AI language models  
- **Razorpay API** — Secure payment gateway integration
